02_sass문법_믹싱문법(mixing)_include기법

1. $mixin기법

웹페이지 전체에 재사용할 css코드를 정의하는 문

    @mixin mixin명{
      //반복해서 사용할 스타일 서식을 지정한다.
    }

2. @include문법

@mixin을 선택자에 포함하는데 사용하는 지시문

    선택자{
      @include mixin명;
    }

믹스인 함수(txt_over)를 사용하여 글자 생략기호 나오게하기

3. @mixin을 중첩하여 새로운 @mixin만들기

    @mixin 믹스인명 A{
      //재사용할 스타일서식
    }

    @mixin 믹스인명 B{
      //재사용할 스타일 서식
    }

    @mixin 믹스인명 C{
      @includ 믹스인명 A;
      @includ 믹스인명 B;
    }
